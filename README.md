## About the Project
 Warehouse manage System.

- This is a central repository for a large company that enables them to control and view the inventory of each of its smaller companiesâ€™ warehouses. This large company can have many smaller companies, and each smaller company may have any number of warehouses. Each of these warehouses have a maximum capacity and cannot surpass that limit. 

- the parent company can the reading, adding, updating, and removal of the items stored in each warehouse and have it displayed on the frontend.
